Site web

Voici un site web que nous avons créé pour une boutique en ligne. Selon les demandes du client, nous avons mis une page Nouveautés et même une catégorie Promotions pour accéder à tous les articles en rabais. Nous avons conçu un site très simple pour ne pas perdre les clients.

Application mobile

Voici une application mobile que nous avons créée pour suivre le nombre de pas effectués dans une journée. Dans l’application, nous avons ajouté des catégories Statistiques, Objectifs et Profil afin que l’utilisateur puisse consulter ses progrès.

Site web sportif

Voici une création que nous avons faite pour une équipe sportive d’hockey. Nous avons mis une page À propos présentant le club et une page de contact pour envoyer des emails et contacter l’équipe. Le site a été conçu de manière simple et dynamique pour représenter l’esprit sportif et la passion du hockey.
